# Focus Clash Solid

GitHub Pages Version ohne Firebase.

## Wichtig
Lade **index.html, style.css, app.js und README.md direkt ins Hauptverzeichnis** deines GitHub-Repositories hoch. Nicht den Ordner `focus-solid` hochladen.

## Start
GitHub > Repository > Settings > Pages > Deploy from branch > main / root.

Dann die GitHub-Pages-URL auf allen Tablets öffnen.
