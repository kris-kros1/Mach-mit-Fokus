# Focus Clash Solid

Stabile, schlichte Multiplayer-Version für GitHub Pages.

## Nutzung
1. Alle Dateien in ein GitHub-Repository hochladen.
2. GitHub Pages aktivieren: Settings → Pages → Deploy from branch → main → /root.
3. Die GitHub-Pages-URL auf allen Tablets öffnen.
4. Ein Tablet: **Spiel hosten**.
5. Andere Tablets: **Beitreten** und Code eingeben.

## Spiele
- Anders: Finde das andere Symbol.
- Merken: Merke markierte Felder.
- Nur Ziel: Reaktion und Impulskontrolle.
- Außen: Periphere Wahrnehmung.
- Gleich: Form und Farbe vergleichen.

## Hinweis
Die Verbindung läuft über WebRTC/PeerJS. Manche Schul-WLANs blockieren WebRTC.
